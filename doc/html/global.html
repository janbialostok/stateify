STATEIFY(parent, isArray) → {Object}

Converts an object into an immutable "state" object which will never be directly manipulated but instead creates a new copy as necessary when a set, delete or get trap is triggered

Parameters:
Name Type Description
parent Object

The object to be converted
isArray Boolean

Specifies that the object being converted is an Array. Must be true in order to properly handle arrays as they will be converted into non-iterable objects if isArray argument is not specified
